# Intelligent-Retail-Insight-System

# Overview
This project is a data-driven retail analytics system that provides insights such as customer segemntation, sales prediction and product recommendations using machine learning techniques.

# Features
-Customer Segemntation 
-Demand prediction
-Market Basket analysis
-Interactive dashboard using Streamlit
